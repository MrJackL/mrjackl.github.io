CocoaPods使用技巧

Sun, Jan 24, 2016

安装和基本使用CocoaPods是十分简单的事情，但是在日常项目当中难免会出现一些奇葩状况，以下列出可能遇到的问题，以及解决办法。

1. 执行pod install 或者 pod update 一直卡住不动，并提示Updating local specs repositories ; (也许是被墙了)这个时候需要执行以下脚本:

pod install --verbose --no-repo-update

或者

pod update --verbose --no-repo-update

2. 使用pod search 或者pod update 时，查询出的第三方库版本低于实际版本； 逐个执行以下脚本:

brew install ruby

sudo gem install cocoapods

pod setup

pod repo update --verbose
