CocoaPods安装

Sun, Jan 24, 2016

  • 介绍

    CocoaPods是iOS中使用最受欢迎的第三方库管理工具，使iOS开发者的对项目的管理变得更加简单；现在有了CocoaPods,就可以通过一段更新脚本就完成了所有第三方类库的更新；而且在Github上有很多开源的项目，大多都支持pods。

    • 前提:Ruby环境

    需要先安装好Ruby环境，这里就不详细介绍了，链接说的很清楚。https://ruby-china.org/wiki/install_ruby_guide

    • 安装CocoaPods

    安装好Ruby环境后，打开Terminal(终端)，理论上只要运行以下脚本就可以安装好CocoaPods:

 sudo gem install cocoapods

但是cocoapods.org被墙了，只好用淘宝的Ruby镜像来访问cocoapods 。

 gem sources --remove https://rubygems.org/

以上脚本执行完成后，再执行以下脚本:

gem sources -a http://ruby.taobao.org/

可以通过以下脚本验证你刚才使用的Ruby镜像是taobao:

gem sources -l

这个时候，你只需要在Terminal中运行：

$ sudo gem install cocoapods

坐等一会，CocoaPods就在你本地下载并且安装好了，大功告成！
